# Bíblia Web App

[Acesse o projeto em produção aqui!](https://sinhorinho.github.io/biblia/)

Este projeto é uma aplicação web simples e responsiva para leitura da Bíblia.

## Funcionalidades

- **Página Inicial Dinâmica**: Uma página inicial intuitiva com um índice completo dos livros da Bíblia, separados em Antigo e Novo Testamento, para fácil navegação.
- **Navegação Aprimorada**:
    - **Seleção Dinâmica**: Escolha livros e capítulos através de seletores dedicados.
    - **Retorno Rápido**: Clique no logo para voltar instantaneamente à página inicial (índice de livros).
- **Leitura Personalizada**:
    - **Exibição Clara**: Leitura otimizada do texto bíblico.
    - **Controles de Fonte**: Ajuste o tamanho da fonte (`+` e `-`) na barra de navegação para conforto visual.
    - **Modo Escuro/Claro**: Alterne entre temas de leitura para diferentes ambientes.
    - **Lembrete de Leitura**: O aplicativo salva automaticamente o último livro e capítulo lido, retomando de onde você parou.
- **Funcionalidade de Busca (BETA)**:
    - **Busca Rápida**: Encontre palavras ou frases em toda a Bíblia através de uma barra de pesquisa dedicada.
    - **Resultados Interativos**: Clique nos resultados da busca para ir diretamente ao livro, capítulo e versículo correspondente, com o verso destacado.
- **Layout Responsivo**:
    - **Barra de Navegação Fixa**: Acesso constante aos controles de navegação e leitura.
    - **Adaptação Móvel**: Design que se ajusta perfeitamente a diferentes tamanhos de tela.


## Tecnologias

- HTML5
- CSS3 (Design responsivo e clean)
- JavaScript (Lógica de navegação e manipulação do DOM)
