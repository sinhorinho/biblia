# Bíblia Web App

Este projeto é uma aplicação web simples e responsiva para leitura da Bíblia.

## Funcionalidades

- **Navegação**: Seleção dinâmica de livros e capítulos.
- **Leitura**: Exibição clara e formatada do texto bíblico.
- **Acessibilidade**: Controles de tamanho da fonte (`+` e `-`) na barra de navegação para facilitar a leitura.
- **Layout Responsivo**:
    - Barra de navegação fixa no topo (`sticky`) para fácil acesso aos controles.
    - Adaptação para dispositivos móveis, com empilhamento vertical dos controles em telas menores.

## Tecnologias

- HTML5
- CSS3 (Design responsivo e clean)
- JavaScript (Lógica de navegação e manipulação do DOM)
